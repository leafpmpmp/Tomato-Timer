# Tomato-Timer
A simple tomato timer, now with a new UI under construction.
## Requirements
Python
```
pip install python
```
PyQt5
```
pip install PyQt5
```
## Run
Just run it with python. If you don't want to see the console window, rename the file ```*.py``` to ```*.pyw``` before executing it.
